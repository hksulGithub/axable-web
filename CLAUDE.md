# Axable.co.kr — Corporate Website

## Stack
- Pure static HTML/CSS/JS (no framework, no build)
- Fonts: Instrument Serif (Google Fonts) + Satoshi (Fontshare)
- Forms: Tally.so (embedded)
- Deploy: serve `/public` directory

## Structure
- `/public/index.html` — Korean landing page (primary)
- `/public/en/index.html` — English version
- `/public/style.css` — Shared stylesheet

## Design
- Dark editorial / tech-noir aesthetic inspired by AIM Intelligence
- Color system: dark background (#07090E–#243044), teal accent (#4DD2A7)
- Typography: Instrument Serif (display/headings) + Satoshi (body)
- Layout patterns: split hero with stats column, asymmetric about grid, timeline methodology, split service cards, multi-column footer
- Scroll reveal animations via Intersection Observer (reveal + stagger-children)
- Fully responsive (desktop/tablet/mobile) with breakpoints at 1024/768/480px

## Brand
- Logo: "ax**able**" (text logo, "able" in accent color)
- Same team and services as AI Future Partners (aifuture.kr)
- Content source: aifuture.kr; UI/design reference: AIM Intelligence
