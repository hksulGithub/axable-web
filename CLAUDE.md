# Axable.co.kr — Corporate Website

## Stack
- Pure static HTML/CSS/JS (no framework, no build)
- Font: Pretendard (CDN)
- Forms: Tally.so (embedded)
- Deploy: serve `/public` directory

## Structure
- `/public/index.html` — Korean landing page (primary)
- `/public/en/index.html` — English version
- `/public/style.css` — Shared stylesheet

## Design
- AIM Intelligence-inspired look: teal accent (#4DD2A7), Pretendard, rounded cards, pill buttons
- Scroll reveal animations via Intersection Observer
- Fully responsive (desktop/tablet/mobile)

## Brand
- Logo: "ax**able**" (text logo, "able" in accent color)
- Same team and services as AI Future Partners (aifuture.kr)
